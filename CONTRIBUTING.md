# Contributing

Thank you for your interest in contributing to Tesla Fullscreen Launcher.

This project is intentionally small, static, and performance-focused. Please keep changes scoped, simple, and aligned with the Tesla in-car browser experience.

## Guidelines

- Do not introduce frameworks or build tooling without prior discussion.
- Keep launcher behavior, UX, and visual design aligned with approved product direction.
- Prefer small pull requests with a clear purpose.
- Test changes in a browser before opening a pull request.
- Do not add analytics, tracking, accounts, or server dependencies.

## Pull Requests

Before submitting a pull request, verify that the static site still works from `index.html` and remains compatible with GitHub Pages.
